About RP Now

RP Now is an open-source web application that allows anyone to conduct a private text-based roleplay with a small group of friends. It is specialized to allow for real-time play as well as for creation of multiple characters.

Technologies

The RP Now back-end is built on Apache, PHP, and MySQL: in other words, a LAMP server.

The front-end is made entirely in HTML5, CSS, and JavaScript — in other words, no browser plugins are required for the user to use this application. All they need is a semi-modern web browser.

Libraries Used

RP Now also uses the following libraries.

  • Slim: Slim is a PHP micro framework that helps you quickly write simple yet powerful web applications and APIs.
  • Twig: The flexible, fast, and secure template engine for PHP
  • jQuery: jQuery is a fast, small, and feature-rich JavaScript library.
  • Spectrum: The No Hassle jQuery Colorpicker
  • Moment.js: Parse, validate, manipulate, and display dates in JavaScript.
